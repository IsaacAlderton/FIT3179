Global CO2 Emissions and Population Statistics

This page contains two embedded visualisations; a world map that displays population density by country, and a bubble plot depicting global CO2 emissions in 2010.

Population Density per Country (2023)

The visualisation on the right shows the population per square kilometre of each country. The world map is displayed using an area-preserving Equal Earth projection, since land area is one of the elements used to compare the population densities of each country. A combination of color hue and value is used to represent the population density attribute. The visualisation includes a tooltip that highlights the country, population density, total population, and percentage of world population.

Global CO2 Emissions (2010)

The Global CO2 emissions bubble plot uses the vertical and horizontal position channels to directly compare 2 quantitative attributes; the population and total CO2 emissions (in Megatons) for each country in the world in 2010. Marker size is further used to represent total emissions per capita, while marker hue and value show the emissions per square km of each country. An interactive slider is incorporated in this visualisation which enables the user to filter countries by their minimum total emissions produced. The tooltip summarises the country, population, emissions per capita and emissions per sqaure kilometre of land area.
